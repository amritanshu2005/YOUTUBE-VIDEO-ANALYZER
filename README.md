# 🚀 YouTube Video Analyzer

An **AI-powered YouTube Video Analyzer** that extracts meaningful insights, summaries, and key information from YouTube videos using **Machine Learning** and **Natural Language Processing (NLP)**.

---

## 📌 Features

* 🎯 Extracts key insights from YouTube videos
* 🧠 Generates concise summaries
* 🔍 Identifies important topics and keywords
* ⚡ Fast and efficient processing
* 🖥️ Simple and user-friendly interface

---

## 🛠️ Tech Stack

* **Python**
* **Machine Learning**
* **Natural Language Processing (NLP)**
* **YouTube Data Processing**

---

## 📂 Project Structure

```
YOUTUBE-VIDEO-ANALYZER/
│── youtube_analyzer.py   # Core logic for analysis
│── ui.py                 # User interface
│── .gitignore            # Ignored files
```

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/amritanshu2005/YOUTUBE-VIDEO-ANALYZER.git
cd YOUTUBE-VIDEO-ANALYZER
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the application:

```bash
python ui.py
```

Then:

* Enter a YouTube video URL
* Get summary and insights instantly

---

## 💡 How It Works

1. Fetches video data from YouTube
2. Processes transcript/content
3. Applies NLP techniques
4. Generates structured insights & summaries
